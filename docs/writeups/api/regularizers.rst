Regularization Functions
========================

.. automodule:: ztlearn.ml.regularizers
  :noindex:

The regularization functions are used to prevent overfitting by modifying the models
objective function by adding additional terms that penalize large weights. This
helps prevents the network from using weights that are redundant.
